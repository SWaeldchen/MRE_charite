mredge_remove_nifti_extension

PURPOSE

Strips file extension from a nifti file, to allow alteration of filenames.

SYNOPSIS

function path_trunc = mredge_remove_nifti_extension(path)

DESCRIPTION 

 Strips file extension from a nifti file, to allow alteration of filenames.

 INPUTS:

 path - a nifti file path, probably from info.ds.subdirs_comps_files

 OUTPUTS:

 truncated pathname
