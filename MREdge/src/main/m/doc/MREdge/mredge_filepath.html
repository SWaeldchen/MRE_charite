mredge_filepath

PURPOSE

Creates consistent file path given directory, frequency and component

SYNOPSIS

function path = mredge_filepath(DIR, f, c)

DESCRIPTION 

 Creates consistent file path given directory, frequency and component

 INPUTS:

   DIR - directory path
   f - frequency in Hz
   c - directional component (1, 2, or 3)

 OUTPUTS:

   text string containing the full path
