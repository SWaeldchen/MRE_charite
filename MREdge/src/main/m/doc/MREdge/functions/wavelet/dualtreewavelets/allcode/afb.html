afb

PURPOSE

Analysis filter bank

SYNOPSIS

function [lo, hi] = afb(x, af)

DESCRIPTION 

 Analysis filter bank

 USAGE:
    [lo, hi] = afb(x, af)
 INPUT:
    x - N-point vector, where
            1) N is even
            2) N >= length(af)
    af - analysis filters
    af(:, 1) - lowpass filter (even length)
    af(:, 2) - highpass filter (even length)
 OUTPUT:
    lo - Low frequecy output
    hi - High frequency output
 EXAMPLE:
    [af, sf] = farras;
    x = rand(1,64);
    [lo, hi] = afb(x, af);
    y = sfb(lo, hi, sf);
    err = x - y; 
    max(abs(err))
