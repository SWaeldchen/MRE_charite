mredge

PURPOSE

Base function for the processing and analysis of MRE data

SYNOPSIS

function outputs = mredge(info, prefs)

DESCRIPTION 

 Base function for the processing and analysis of MRE data

 INPUTS:

   info - MREdge acquisition info structure generated with mredge_acquisition_info
   prefs - MREdge preferences structure generated with mredge_prefs

 OUTPUTS:

   returns a structure containing outputs as specified by prefs

 EXAMPLE USAGE:

   local_data_path = pwd; % copy raw data locally before running MREdge!
   info = mredge_info('path', local_data_path, 'magnitude', 2, 'phase',
   3);
   prefs = mredge_prefs('outputs.elastograms', 1);
   elastograms = mredge(info, prefs);

CROSS-REFERENCE INFORMATION

This function calls:
  • mredge_clean_acquisition_folder Cleans all processing files from local acquisition folder, leaving only DICOMs.
  • mredge_dicom_to_nifti Converts raw DICOM files into NIfTI objects, which are used for all MREdge processing
  • mredge_load_FT_as_5d Load Fourier-transformed wavefield into MATLAB
  • mredge_load_amplitudes Load amplitude maps into MATLAB
  • mredge_load_elastograms Load elastograms into MATLAB
  • mredge_organize_acquisition Creates nested folder structure for NIfTI files
  • mredge_average_magnitude Creates a single, averaged t2 magnitude map from an MRE acquisition, the preferred volume to use for co-registration and segmentation.
  • mredge_bandpass Applies highpass and lowpass butterworth filters if they are set in prefs
  • mredge_denoise Denoise Fourier-transformed wavefield
  • mredge_denoise_raw Denoises raw complex data prior to phase unwrapping
  • mredge_distortion_correction Correct for distortion in the MRE acquisition using FSL or SPM libraries
  • mredge_invert_and_stats Performs wave inversion and inversion-related statistical measures
  • mredge_motion_correction Performs motion correction on raw MRE acquisitions, using FSL or SPM
  • mredge_phase_unwrap Applies phase unwrapping methods
  • mredge_remove_divergence Numerical divergence removal of the wave field
  • mredge_remove_ipds Removes interslice phase discontinuities
  • mredge_set_environment Set environment variables for MREdge
  • mredge_slice_align Dejitters slices using adaptation of Nikolova et al 2001 dejitter algorithm
  • mredge_temporal_ft Temporally Fourier-transforms the complex wave field
  • mredge_snr Calculates SNR maps and records SNR values in stats folder
